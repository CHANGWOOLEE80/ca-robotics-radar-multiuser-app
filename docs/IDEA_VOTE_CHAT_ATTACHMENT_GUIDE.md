# Idea Board Voting & Team Chat Attachment Guide

## 1. Idea Board 좋아요/투표 기능

이번 버전부터 Idea Board의 `+ 투표` 기능은 SNS의 좋아요 기능과 유사하게 동작합니다.

### 주요 변경사항

- 사용자 1명은 하나의 Idea에 1회만 좋아요를 누를 수 있습니다.
- 이미 좋아요를 누른 사용자가 다시 클릭하면 좋아요가 취소됩니다.
- 동일 사용자가 반복 클릭해도 투표수가 계속 증가하지 않습니다.
- 개인별 계정 기준으로 투표 여부를 판단합니다.

### 운영 기준

- 반드시 개인별 Knox ID 계정으로 로그인하여 사용하십시오.
- 공용 계정을 사용할 경우 사용자별 투표 관리가 불가능하므로 권장하지 않습니다.
- 기존에 입력되어 있던 투표수는 `baseVotes`로 유지되고, 신규 개인 투표가 추가됩니다.

## 2. Team Chat 이미지/동영상 첨부 기능

Team Chat에서 메시지와 함께 이미지 또는 동영상 파일을 업로드할 수 있습니다.

### 지원 파일

- Image: JPG, PNG, GIF, WEBP
- Video: MP4, WEBM, MOV

### 제한 기준

- 메시지 1건당 최대 3개 파일
- 파일 1개당 최대 25MB
- 대용량 동영상은 필요한 부분만 짧게 편집 후 업로드 권장

### 저장 위치

업로드 파일은 서버 PC의 아래 폴더에 저장됩니다.

```text
uploads
```

Team Chat 메시지에는 파일 자체가 아니라, 업로드 파일 경로와 메타정보가 저장됩니다.

## 3. 백업 시 유의사항

기존에는 `data` 폴더만 백업하면 되었지만, 이번 버전부터는 Team Chat 첨부파일이 `uploads` 폴더에 저장됩니다.

따라서 운영 백업 시 아래 두 폴더를 함께 백업하십시오.

```text
data
uploads
```

## 4. 보안 유의사항

- 회사 보안자료, 고객 기밀자료, 개인정보가 포함된 이미지·동영상은 업로드하지 마십시오.
- 기술 검토용 참고 이미지, 장비 사진, 공개 가능 영상 중심으로 사용하십시오.
- 공식 증빙자료는 기존 사내 문서관리 기준에 따라 별도로 관리하십시오.
