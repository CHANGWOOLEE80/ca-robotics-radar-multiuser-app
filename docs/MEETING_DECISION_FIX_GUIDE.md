# Meeting Notes / Decision Log 활성화 수정 안내

## 수정 내용

이번 수정 버전에서는 Productivity Center에서 추가된 Meeting Notes와 Decision Log가 정상적으로 열리고 목록이 표시되도록 다음 항목을 보완했습니다.

1. Meeting Notes 전용 Table Header 추가
2. Meeting Notes 전용 Row Template 추가
3. Decision Log 전용 Table Header 추가
4. Decision Log 전용 Row Template 추가
5. Action Items 목록 표시용 Header/Row Template 추가
6. Meeting Notes 목록에서 `Action 전환` 버튼이 동작하도록 유지

## 사용 방법

- Productivity Center에서 `+ Meeting Note` 또는 `+ Decision`을 눌러 신규 등록할 수 있습니다.
- 상단 메뉴의 `Meeting Notes`, `Decision Log`를 눌러 전체 목록을 확인할 수 있습니다.
- Meeting Notes에서 `Action Items` 필드에 줄 단위로 작성한 후 `Action 전환` 버튼을 누르면 Action Item으로 등록됩니다.

## 기존 데이터 유지

기존 운영 폴더의 아래 항목을 새 버전 폴더에 복사하면 기존 데이터를 유지할 수 있습니다.

```text
data
uploads
```
