# Team Chat @ Mention 자동완성 기능

## 목적
Team Chat에서 매번 Knox ID를 직접 입력하지 않아도 `@`만 입력하면 등록된 그룹원 목록을 표시하여 빠르게 호출할 수 있도록 개선했습니다.

## 사용 방법
1. Team Chat 입력창에서 `@`를 입력합니다.
2. 등록된 그룹원 목록이 표시됩니다.
3. 이름 또는 ID 일부를 계속 입력하면 목록이 필터링됩니다.
   - 예: `@js` → `js1978.kim` 표시
   - 예: `@ab` → `ab.sharafat` 표시
4. 원하는 사용자를 마우스로 클릭하거나 키보드로 선택합니다.
   - `↑ / ↓`: 선택 이동
   - `Enter` 또는 `Tab`: 선택 확정
   - `Esc`: 목록 닫기
5. 선택한 사용자는 `@KnoxID` 형식으로 메시지에 삽입됩니다.

## 표시 기준
- 등록된 사용자 전체가 표시됩니다.
- 현재 접속 중인 사용자는 `online`으로 표시됩니다.
- 미접속 사용자도 `registered` 상태로 표시되며, 멘션은 메시지에 기록됩니다.
- 알림은 상대방이 App에 접속 중일 때 화면 안에서 표시됩니다.

## 주의사항
- 브라우저 알림은 사용자가 알림 허용을 해야 표시됩니다.
- 상대방이 접속해 있지 않은 경우 실시간 알림은 가지 않지만, Team Chat 메시지와 멘션 기록은 남습니다.
