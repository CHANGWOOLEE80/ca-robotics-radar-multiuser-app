# Presence & @Mention Guide

## 1. 현재 접속자 표시

Team Chat 좌측 패널에 현재 접속 중인 사용자가 표시됩니다.

- 접속자는 로그인된 사용자 기준으로 표시됩니다.
- 같은 사용자가 여러 브라우저 탭으로 접속해도 사용자 단위로 1명처럼 보입니다.
- 상단 Live 표시에는 현재 온라인 인원 수가 함께 표시됩니다.

## 2. @호출 기능

Team Chat 메시지에서 아래 형식으로 입력하면 해당 사용자에게 알림이 표시됩니다.

```text
@carlos.lee
@js1978.kim
@shb.bae
@ab.sharafat
@ygiz.lee
@hs98.choi
```

예시:

```text
@js1978.kim 이 Vendor 자료 검토 가능할까요?
```

## 3. 온라인 사용자 버튼으로 호출

Team Chat 좌측의 현재 접속자 목록에서 사용자를 클릭하면 메시지 입력창에 자동으로 `@KnoxID`가 삽입됩니다.

## 4. 알림 방식

상대방이 App을 열어 두고 있으면 다음 알림이 표시됩니다.

- 화면 우측 하단 Toast 알림
- 상단 사용자 Badge에 멘션 수 표시
- 브라우저 탭 제목 깜빡임
- 브라우저 알림을 허용한 경우 데스크톱 알림 표시

브라우저 알림은 Team Chat 좌측의 `브라우저 알림 허용` 버튼으로 활성화할 수 있습니다.

## 5. 제한사항

- App이 닫혀 있거나 PC가 꺼져 있으면 실시간 알림은 표시되지 않습니다.
- 브라우저 또는 Windows 알림 정책이 차단되어 있으면 데스크톱 알림이 표시되지 않을 수 있습니다.
- 사내 로컬 PC 서버 기반이므로, 알림은 App 접속 중인 사용자에게만 유효합니다.

## 6. 권장 사용법

중요한 호출은 아래와 같이 명확하게 작성하는 것이 좋습니다.

```text
@ab.sharafat Humanoid Module Assembly 관련 논문 Reference 확인 부탁드립니다.
@ygiz.lee Air Dome Vendor Clarification 항목 추가 검토 부탁드립니다.
```
